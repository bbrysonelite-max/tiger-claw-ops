# Tiger Bot Onboarding & Flywheel Website

Interactive single-page application visualizing the complete Tiger Bot system — from onboarding interviews through the 5-stage flywheel, lead generation, nurture campaigns, regional intelligence, provisioning pipeline, and webhook setup.

## Stack
- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion (animations)
- shadcn/ui components

## Sections
1. **Onboarding Interviews** — Interview 1 (Who Are You?) & Interview 2 (Ideal Customer Profile)
2. **API Key Rotation** — 72-hour free trial → customer key rotation with provider guides
3. **Flywheel Overview** — 5-stage animated cycle (Discovery → First Contact → Nurture → Convert → Retain)
4. **Flywheel Explainer** — Deep-dive with Funnel vs. Flywheel comparison and interactive diagram
5. **Lead Generation** — Platform sources and outreach channels
6. **Nurture Campaigns** — 10-touch sequences for 10 gig economy roles using 10 psychology principles
7. **Why It Works** — Cialdini's principles of persuasion explained
8. **Aftercare** — Post-conversion retention system
9. **Regional Intelligence** — Platform-specific lead sources for 5 global regions with language flags
10. **Provisioning Pipeline** — Automated paid/trial bot provisioning flow
11. **Webhook Setup Guide** — Step-by-step Stripe & Stan Store webhook configuration

## Local Development
```bash
pnpm install
pnpm dev
```
